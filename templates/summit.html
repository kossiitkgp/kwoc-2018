Tutorials

Introduction to Python

Objective: Familiarity with syntax and simple language features

Contents/Syllabus: Types, Variables, Strings, Boolean and Logic, Data Structures (List, Tuple, Dictionary), Looping, Functions

Resource:

  • https://github.com/OrkoHunter/KOSS-Python-classes

Introduction to Git/GitHub

Objective: Introduction to VCS, commonly used Git commands and GitHub workflow

Resource:

  • 1. Introduction to linux command line - Udacity (https://in.udacity.com/course/linux-command-line-basics--ud595)
  • 2. Git and GitHub - Udacity (https://in.udacity.com/course/how-to-use-git-and-github--ud775)
  • 3. Git Interactive Practice(https://learngitbranching.js.org/)

Introduction to Javascript and Asynchronous programming

Objective: Intro to basic functionality of the language, Basics of Event Listeners and Callbacks.

Contents/Syllabus: Language familiarity | Working on Browser Console | Node.js

Resource:

  • https://github.com/kossiitkgp/JavaScript-Classes

Python Advanced

Objective: Hands-on approach to programming in Python

Contents/Syllabus: Magic methods in Object Oriented, Packaging, Import System

Resource:

  • https://github.com/OrkoHunter/KOSS-Python-classes
  • https://developers.google.com/edu/python/

Pragmatic GoLang

Objective: Overview of the language features with focus on concurrency.

Contents/Syllabus: Basic syntax | Type System | User I/O | Intro to Concurrency and Parallelism: (i) Channels (ii) select` statement | Building a fake search engine from the above concepts

Resource:

  • https://github.com/kossiitkgp/GoLang-Classes
